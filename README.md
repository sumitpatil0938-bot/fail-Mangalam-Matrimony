# Fail-Mangalam-Matrimony

## About Project

Mangalam Matrimony was my personal web development project inspired by matrimonial websites.

The goal was to create a complete matrimony platform where users could:

- Create accounts
- Login securely
- Sign in using Google Authentication
- Search for matches
- View profiles
- Manage their matrimony account

## Problems Faced

This project was not completed because of several challenges:

- Firebase Authentication errors
- Unauthorized Domain issues
- Google Login configuration problems
- Page routing and redirection issues
- Project structure management difficulties
- Lack of backend/database implementation
